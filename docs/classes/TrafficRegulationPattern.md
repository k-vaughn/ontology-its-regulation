# The Traffic Regulation Pattern of the Traffic Regulation Ontology

This ontology defines concepts related to traffic regulations.

This pattern imports the following files:

- [https://w3id.org/itsdata/regulation/v1/ConditionPattern](https://w3id.org/itsdata/regulation/v1/ConditionPattern)
- [https://w3id.org/itsdata/regulation/v1/Core](https://w3id.org/itsdata/regulation/v1/Core)

This pattern consists of the following classes:

- [Implementation Status Code](ImplementationStatusCode.md)
- [Reason Code](ReasonCode.md)
- [Regulatory Scope Code](RegulatoryScopeCode.md)
- [Rule Maker Role](RuleMakerRole.md)
- [Traffic Regulation](TrafficRegulation.md)
- [Traffic Regulation Order](TrafficRegulationOrder.md)
- [Type Of Regulation](TypeOfRegulation.md)
This module defines the following properties:

- [associatedTrafficControlDevice](../properties/associatedTrafficControlDevice.md)
- [authorizingTrafficRegulation](../properties/authorizingTrafficRegulation.md)
- [hasDetails](../properties/hasDetails.md)
- [implementationStatus](../properties/implementationStatus.md)
- [implementedLocation](../properties/implementedLocation.md)
- [implementedValidity](../properties/implementedValidity.md)
- [issuedTro](../properties/issuedTro.md)
- [issuingAuthority](../properties/issuingAuthority.md)
- [locationByOrder](../properties/locationByOrder.md)
- [memberTrafficRegulation](../properties/memberTrafficRegulation.md)
- [reasonCode](../properties/reasonCode.md)
- [regulatoryScope](../properties/regulatoryScope.md)
- [validityByOrder](../properties/validityByOrder.md)


The formal definition of this pattern is available in TURTLE Syntax in two files, the [core semantics](../regulation-pattern.ttl) and the SHACL [restrictions](../regulation-shacl.ttl).
